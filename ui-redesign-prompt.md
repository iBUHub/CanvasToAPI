# CanvasToAPI 前端界面重设计提示词

## 项目背景

CanvasToAPI 是一个代理服务器，它将 Gemini Canvas 的网页界面包装成 API 端点，使其兼容 OpenAI、Gemini 和 Anthropic 的 API 格式。系统使用 Playwright 浏览器自动化与 Canvas 网页交互，并将 API 请求转换为浏览器操作。

## 前端网页用途

这个前端界面是 CanvasToAPI 的管理面板，供用户：

1. 监控代理服务器的运行状态（连接状态、活跃会话数、各账号负载情况等）
2. 管理浏览器会话和认证账号
3. 配置运行时参数（如最大并发上下文数、会话路由策略、日志级别等）
4. 查看系统日志和错误信息

## 技术要求

- 使用 Vue.js 3 + Element Plus + Vue Router + Vite
- 支持国际化（提供中英文切换能力）
- 支持深色/浅色主题切换

## 设计要求

请自由发挥设计，无需遵循任何特定的设计指引或风格约束。你可以完全重新构思布局、配色、交互方式和视觉风格，创造一个你认为最适合这个工具的管理界面。
